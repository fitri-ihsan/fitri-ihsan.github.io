Panduan Penggunaan

1

Input Manual & Copy Paste

Klik tombol + di pojok kanan bawah.

Anda bisa mengetik satu per satu atau Copy-Paste langsung dari Excel/Word dengan format:

Nama Tamu 1 - Alamat Tamu 1
Nama Tamu 2 - Alamat Tamu 2
Nama Tamu 3 - Alamat Tamu 3

Keterangan: Gunakan tanda strip (-) sebagai pemisah antara Nama dan Alamat.

2

Import File Excel (.xlsx)

Klik tombol "Import Excel" di menu atas. Siapkan file Excel dengan susunan kolom:

  • Kolom A: Nama Tamu
  • Kolom B: Alamat / Jabatan / Kota

Catatan: Jika baris pertama berisi tulisan "Nama" atau "Name", baris tersebut akan otomatis diabaikan (dianggap Header).

3

Edit, Hapus & Reset

  • Edit Data: Arahkan kursor ke label (atau tap pada HP), lalu klik icon pensil di pojok label.
  • Hapus Data: Di dalam menu Edit, klik tombol merah "Hapus".
  • Reset Total: Klik tombol merah "Reset" di bagian atas aplikasi untuk menghapus SEMUA data dan memulai dari nol.
4

Sinkronisasi Google Sheet (Opsional)

Fitur ini untuk menyimpan data secara online (Cloud). Langkah setup:

  1. Buat Google Sheet baru di sheets.google.com.
  2. Klik menu Extensions > Apps Script.
  3. Hapus kode bawaan, lalu copy-paste kode di bawah ini.
  4. Klik Deploy (Biru) > New Deployment.
  5. Pilih type: Web App. Akses (Who has access): Pilih Anyone.
  6. Deploy, lalu Copy Web App URL.
  7. Paste URL tersebut di menu Pengaturan (Icon Gear) aplikasi ini.
