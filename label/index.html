Panduan Lengkap

1. Input Manual / Copy-Paste

Klik tombol (+) di bawah. Paste data dari Excel dengan format Nama - Alamat.

2. Import File Excel

Klik tombol Import Excel. Gunakan file .xlsx dengan Kolom A: Nama dan Kolom B: Alamat.

3. Download Word

Klik tombol Word untuk mendownload file .docx yang siap cetak. Ukuran akan menyesuaikan pengaturan Tampilan yang Anda pilih.

4. Google Sync (Opsional)

Untuk menyimpan data online, pasang Google Apps Script di Google Sheet Anda dan masukkan URL Web App di menu Pengaturan.
